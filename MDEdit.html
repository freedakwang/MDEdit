AI 助手设置

配置 OpenAI 兼容的 API 参数。支持 OpenAI、DeepSeek、通义千问、Moonshot 等。

示例：https://api.openai.com/v1/chat/completions
DeepSeek: https://api.deepseek.com/v1/chat/completions
通义千问: https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
自定义: https://your-domain/v1/chat/completions
你的 API 密钥，仅保存在浏览器本地，不会上传到任何服务器
输入模型名称或从下拉列表中选择
AI 单次回复的最大 token 数量，建议 2048-8192
值越高回复越有创意，值越低越精准。范围 0-2，建议 0.5-1.0
发送给 AI 的历史对话 + 当前消息的总 token 数上限。超出时自动裁剪最早的历史消息。建议设为模型上下文窗口的 60%-80%
如果遇到 CORS 跨域错误，可填写代理地址。示例：
https://corsproxy.io/? 或 https://api.allorigins.win/raw?url=
留空则直接请求 API 地址（需服务器支持 CORS）
释放文件以打开（支持 MD/PDF/Word/Excel/图片）
